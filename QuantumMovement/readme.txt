Solve 2D Schrodinger Equation in a customizable setting.

CrankNicolson folder contains:
    - Math and Physics: Crank Nicolson solver, functions/methods to calculate energy, etc.
    - Animation: Matplotlib module. Creates animation given quantum system. This animation can be shown
      or saved, and interactivity can be added here.
    - Principal file is for testing purposes. The main program will be outside, and kivy too.

To run the main program run:
    -   main.py
